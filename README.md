# BHC - Bootcamp Hackathon COBOL

Repositório com os exercícios desenvolvidos durante o Bootcamp Hackathon COBOL.

Aqui você vai encontrar programas em COBOL evoluindo de conceitos básicos (entrada de dados, variáveis, condições) até tópicos mais avançados, como tabelas, subprogramas e processamento de arquivos.

Para mais informações, na pasta do desafio e dos exercícios está um README direto explicando sobre o assunto.
